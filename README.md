# Zero2Stocks
Zero2Stocks is a stock market portfolio management backend built with a clean, scalable multi-layered .NET architecture.
The system allows users to:

Add/remove stocks to their portfolios

Track quantities and average buy prices

Organize multiple portfolios

Add comments to assets

Manage user authentication

Build a real stock-tracking workflow

This project demonstrates real-world backend development, following SOLID and Domain-Driven Design principles.

Tech Stack:

-.NET 8

-ASP.NET Web API

-Entity Framework Core

-SQL Server / PostgreSQL

-AutoMapper

-Repository Pattern & Dependency Injection


Future Improvements:

-Frontend integration (Vue.js)

-Real-time stock data provider integration

-Profit/Loss automatic calculation

-Watchlist module

-Portfolio performance charts
